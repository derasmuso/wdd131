Siteplan - Food Recipes

Site Name - Flavors At Home

This name represents a website that is focused on food recipes for home cooks.

Domain Name: flavorsathome.com

Site Purpose

The site provides food recipes, a newsletter that users can subscribe to, and a way for users to submit their own recipes to be reviewed with a possibility of them being added to the site.

Scenarios

    What recipe only uses 5 ingredients?
    What recipe takes only 30 minutes to cook?
    How can I leave a review or tip for a recipe?
    I have a recipe to share, how can I share this recipe with others on the website?
    How can I contact the website owners?

Color Schema

Primary Color: #8C1F17 (buttons and headers)

Secondary Color: #70371A (color blocks and embellishments)

Secondary Color: #231815 (text headers)

Accent Color: #F3E6D9 (background color)

Typography

Body Font: "Roboto", sans-serif
